Etiquetas Semánticas Estructurales

Es una etiqueta contenedora que no tiene semántica.
<header> - Cabecera de un sitio Web o de una sección.
<main> - Define la sección principal del documento. Solamente puede existir UNA etiqueta "main" por documento.
<footer> - Pie de página de un sitio Web o de una sección.
<nav> - Sirve para agregar el menú de navegación del sitio Web.
<article> - Representa una sección de autocontenido(es decir, una sección que por sí sola se explica. Por ejemplo: un artículo de un blog; dicho artículo, se incluiría dentro de la etiqueta "article".
<aside> Contenido complementario o secundario (relacionado con el contenido principal). También se puede incluir dentro de esta etiqueta la publicidad del sitio Web. Por ejemplo: 
            
                <main>
                    <h1>Publicación del Blog</h1>
                    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec viverra nec nulla vitae mollis.</p>
                    <p>etc.</p>
                </main>
                  
                  <aside>
                    <h3>Acerca del autor</h3>
                    <p>John Smith - FullStack Developer.</p>
                  </aside>
<section> - Representa una sección de contenido. IMPORTANTE: dependiendo del contenido de la página Web, puede ser que se halle una etiqueta "article" dentro de una etiqueta "section", o bien que haya una etiqueta "section" dentro de una etiqueta "article".
<address> - Representa una dirección.
